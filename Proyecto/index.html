Software Engeneering


Software engineering is a discipline made ​​up of a set of methods, tools and techniques used in the development of computer programs (software).


This discipline transcends programming activity, which is the cornerstone when building an application. The software engineer is responsible for the overall management of the project so that it can be developed within a specified time and within budget.


Software engineering, therefore, includes the previous analysis of the situation, the project design, software development, testing to confirm proper operation and implementation of the system.
